# State Machine Example

Demonstrates state machines, transitions, and event dispatching.

## Features

✅ Multiple states
✅ Event dispatching
✅ State transitions
✅ Current state display
✅ Guard conditions

## Running

Just open `index.html` in a browser (no build required).

## How It Works

The example has a state machine with three states:
- **idle** — Initial state
- **state1** — Transition via `event1`
- **state2** — Transition via `event2`

Click buttons to dispatch events and watch the state machine transition between states.

Open the browser console to see state change events logged.
