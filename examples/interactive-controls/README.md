# Interactive Controls Example

Demonstrates playback controls, timeline scrubbing, and speed adjustment.

## Features

✅ Play/pause/stop buttons
✅ Timeline slider (seek)
✅ Speed control
✅ Time display
✅ Event logging

## Running

Just open `index.html` in a browser (no build required).

## How It Works

- **Play/Pause/Stop** — Control playback state
- **Speed Slider** — Adjust playback speed (0.25x - 2x)
- **Timeline Slider** — Seek to specific positions
- **Time Display** — Shows current time and duration

Open the browser console to see events being logged.
