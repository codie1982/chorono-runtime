Chorono Runtime - Asset & Media Handling

Asset Types Supported

  • imagePNG, JPG, WebP, GIF
  • videoMP4, WebM, OGG
  • audioMP3, WAV, OGG
  • fontTTF, OTF, WOFF, WOFF2
  • svgScalable Vector Graphics
  • lottieLottie Animation
  • spriteSheetGrid-based Frame Animations

Asset Storage Modes

  • Embedded: Base64-encoded in scene JSON
  • External: URL to CDN or server
  • Project-relative: Part of exported scene package
  • Remote: Loaded from remote URL at runtime

Best Practices

  • Preload assets before playback for smooth animation
  • Use external assets for large files (videos, high-res images)
  • Embed small assets to reduce HTTP requests
  • Optimize image formats (WebP for modern browsers)
  • Handle loading errors with fallbacks
